There are three folders under this project solution "MeetingBooking", "MeetingBooking.Business" and "MeetingBooking.Test".

"MeetingBooking" - MVC Web Application
"MeetingBooking.Business" - Business layer
"MeetingBooking.Test" - Unit test



Steps:
1.Open the solution in visual studio 
2.Change the Log Path in the “appsettings.json”  of the "MeetingBooking" project.
3.Run the "MeetingBooking" project.
